---
title: "How to earn skills?"
---

Verified Skills on GoodFreshers provide trusted proof of a candidate’s abilities. These skills help candidates demonstrate their capabilities with credibility and improve their visibility to recruiters.

All verified skills are stored and managed in the **Skill Repository**.

Candidates can earn verified skills through the following methods:

## 1. Source-Verified Skills (Projects / Work Experience)

Candidates can validate skills gained through real-world projects or professional experience.

### Process

- Add a project or work experience to the profile
- Select the relevant skills applied during the work
- Initiate a verification request
- Provide the official business or organization email ID of the verifier
- Personal email IDs (e.g., Gmail, Yahoo) are not accepted
- The platform validates the email domain
- A verification request is sent to the provided contact
- Skills are marked as verified upon approval

### Outcome

- Verified skills are added to the Skill Repository
- These skills carry high credibility as they are validated by a professional contact

## 2. Certificate \+ Assessment Verification

Candidates can validate skills by combining certification proof with an assessment.

### Process

- Upload a valid certificate
- Map the certificate to relevant skills
- Complete the associated assessment
- Achieve a passing score

### Outcome

- Skills are verified based on assessment performance
- Stored in the Skill Repository
- Ensures both proof of completion and validation of knowledge

## 3. YouTube Learning \+ Assessment

Candidates can convert structured learning content into verified skills.

### Process

- Add an educational YouTube video or playlist
- Complete the content within the platform
- Take the associated assessment
- Achieve a passing score

### Outcome

- Skills are verified upon successful completion
- Added to the Skill Repository
- Enables candidates to validate self-paced learning

## Additional Notes

- Verified skills are clearly distinguished within the candidate profile
- Recruiters can rely on verified skills as trusted indicators of capability
- Candidates can use multiple methods to build a comprehensive skill profile
- A stronger set of verified skills improves discoverability and hiring outcomes
