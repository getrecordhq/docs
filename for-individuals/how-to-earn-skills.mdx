## Overview

Skill Verification enables candidates to earn trusted, verifiable skill badges by proving their expertise through either:

- Human Endorsement
- Assessment Verification

Every verified skill is stored inside the **Skill Repository**, where candidates can view their verification status, trust level, verification proofs, touchpoints, and supporting evidence.

The verification system is designed to help recruiters confidently evaluate a candidate's capabilities using real-world validation rather than self-declared skills.

---

# Verification Methods

Candidates can verify skills using two methods:

## 1. Human Endorsement

A professional contact validates the candidate's skill based on actual working experience.

Examples:

- Reporting Manager
- Functional / Technical Lead
- Client / Stakeholder
- Founder / CXO
- Peer / Colleague

Human endorsements generate high-trust verification signals because they are validated by real individuals who have directly observed the candidate's work.

### Key Rules

- Up to **5 skills** can be bundled in a single endorsement request.
- Only **official company email addresses** are accepted.
- Personal email addresses such as Gmail, Yahoo, Outlook, etc. are not allowed.
- One source must be selected during verification.
- The verifier receives a secure verification request.
- Skills become verified once approved by the verifier.

---

## 2. Assessment Verification

Candidates can verify a skill by successfully completing an AI-generated assessment.

The assessment is dynamically generated based on the selected skill and source.

### Key Rules

- Only **1 skill** can be assessed at a time.
- Assessment questions are generated from:
  - Work Experience
  - Projects
  - Certifications
  - Other Activities
- Candidates must achieve the required passing score.
- Verified skills are automatically added to the Skill Repository.

| Endorsement Type | Trust Level |
| :-- | :-- |

## Step 4 — Select Verification Source

The candidate selects the source where the skill was demonstrated.

### Available Sources

### Certifications

Skills supported through certifications.

### Other Activities

Skills demonstrated through extracurricular or professional activities.

## Step 6 — Bundle Additional Skills

Human endorsement allows candidates to verify multiple skills together.

### Bundle Rules

- Maximum 5 skills per endorsement request.
- All bundled skills must relate to the selected source.
- Each skill may include supporting proof.

### Examples

For a Work Experience source:

- TypeScript
- JavaScript
- React
- Node.js
- API Development

A single manager can validate all five skills through one request.

### Endorser Information

- Email

### Selected Source

- Project
- Certification

# Verifier Experience

The verifier receives a secure verification link.

The verifier can review:

### Candidate Information

- Name
- Email
- Verification ID

### Reference Source

- Work Experience
- Project
- Certification

### Skills Requested

Each skill can be:

- Approved
- Not Sure

### Final Confirmation

Verifier confirms:

> "The information accurately reflects the candidate's skills and experience."

Once submitted:

- Approved skills become verified.
- Touchpoints are added to the skill record.
- Trust score is updated.

Skill becomes verified.

### Fail

### Paused

# Verification Touchpoints

Touchpoints represent independent verification proofs accumulated against a skill.

A skill becomes stronger as additional touchpoints are added.

---

## Available Verification Proofs

Each skill can accumulate up to 6 independent proofs.

### Human Endorsement Proofs

1. Reporting Manager
2. Functional / Tech Lead
3. Client
4. Founder / CXO
5. Peer

### Assessment Proof

1. Assessment Verification

---

## Maximum Verification Capacity

### Per Source

- Maximum 10 unique skills

### Per Skill

- Maximum 6 proofs

### Maximum Touchpoints Per Source

```text
10 Skills × 6 Proofs = 60 Touchpoints
```

# Skill Repository

All verified and in-progress skills are stored in the Skill Repository.

The repository provides a complete history of verification activity.

---

## Skill Statuses

### Verified

Skill has received successful verification.

### In Progress

Verification request has been sent and is awaiting completion.

### Pending Review

Awaiting human verifier response.

### Assessment Pending

Assessment has not yet been completed.

### Failed

Assessment completed but passing score not achieved.

### Expired

Verification validity period has ended.

# YouTube Learning Verification

Transform your self-learning into verified, recruiter-recognized skills.

The YouTube Learning Verification feature allows candidates to earn skill badges by completing educational content and successfully demonstrating their knowledge through an assessment. Skills earned through this method are added directly to the Skill Repository, alongside other verified skills earned through endorsements and assessments.

Each verified skill includes verification metadata, learning history, assessment outcomes, and trust signals that help recruiters understand how the skill was earned. Candidates can also receive a Certificate of Completion, providing tangible proof of their learning journey.

This enables learners to convert educational YouTube content into credible, shareable professional achievements while building a stronger and more discoverable skill profile.

# Edge Cases & Validation Rules

## Human Endorsement Rules

| Scenario | Allowed | Behavior |
| :-- | :-- | :-- |
| Same Skill \+ Same Source \+ Same Endorsement Type | ❌ | Blocked |
| Same Skill \+ Same Source \+ Different Endorsement Type | ✅ | Allowed |
| Bundle Contains Already Verified Skill (Same Type) | ❌ | Remove Skill |
| Bundle Contains Already Verified Skill (Different Type) | ✅ | Allowed |
| Duplicate Skill Within Same Source | ❌ | Blocked |
| More Than 10 Skills In A Source | ❌ | Blocked |

---

## Assessment Rules

| Scenario | Allowed | Behavior |
| :-- | :-- | :-- |
| Same Skill \+ Same Source Already Passed | ❌ | Retake Blocked |
| Failed Assessment | ✅ | Retake Allowed |
| Paused Assessment | ✅ | Resume Available |
| Resend Verification | ✅ | Link Can Be Resent |
