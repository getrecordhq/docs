## Adding a Project

Go to **Profile → Projects & Activities → Add Project** and fill the fields below.

## Project Name

Enter a clear name for your project.

**Examples:** Personal Portfolio Website, Loan EMI Calculator App, Food Delivery App UI.

<Info>
  A simple, professional name helps recruiters understand the project at a glance.
</Info>

## Project Type

Choose whether you worked **Individually** or as part of a **Collaborative** team.

- **Individual** → You completed the project alone
- **Collaborative** → You worked with a team (classmates, colleagues, office-related etc.)

## Associated With

Choose where this project was done.

- If you have **verified education** or **verified work experience**, those institutions/companies will automatically appear here
- Select the relevant one (e.g., “Coimbatore Institute of Technology (Student)”, “ABC Private Limited (Intern)”)

If the project is not linked to an entry in your profile, choose **Others** and manually enter the association.

<Info>
  This helps recruiters understand the project context.
</Info>

## Project Cover Image

Upload a cover image or screenshot that visually represents your project.

Recommended: **1600px width**, JPG/PNG format.

<Info>
  A good cover image makes your project stand out.
</Info>

## Start Date & End Date

Select when you started and finished the project.

If you are still working on it, enable **“I’m currently working on this project.”**

Dates help recruiters understand your project timeline and consistency.

## Project Link

Add a link where your project can be viewed.

**Examples:**

- GitHub Repo
- Behance / Dribbble
- Notion / Website
- Google Drive (public link)

**Note:** Make sure the link is accessible.

## Description

Write a short overview of your project what you built, how you approached it, and what impact it created.

**Include:** Describe the problem you solved, your role in it, the tools/technologies you used, and the key learnings or outcomes.

**Character limit:** 500 characters (keeps it short and readable) Use **AI Rephrase** to make your description more polished and professional instantly.

## Add Skills

Add up to **5 skills** used in this project.

**Examples:** Figma, React, User Research, Python, Data Analysis.

If a skill is not available:

- Click **Request New Skill**
- Our team reviews and approves valid skills
- They’ll be added to the Skill Repository shortly

<Info>
  Skills make your project more searchable and strengthen your profile.
</Info>

## Project Verification

You can request verification to build trust and credibility.

### How it works

1. Click **Get verified**
2. Enter the official email of your:
   - Mentor
   - Professor
   - Client
   - Manager
   - Founder
3. They receive a verification link through email.
4. Once approved, your project is marked as **Verified**

### Important

- Use **organizational/institutional email IDs**
- Ask the verifier to check **Spam/Promotions** if the mail isn't found

<Info>
  Verified projects show proof of your work and improve your chances with recruiters.
</Info>
